## Image Auto Sync to OSS
需求环境

> wordpress > 5.0
> php > 7.0

将文章内非OSS图片自动上传到OSS并替换图片地址，清理本地媒体库(可选)；

Pro 版支持一键替换历史文章内图片。

使用前请备份数据库。
